# Rental Vehicle System

A web-based rental platform built using PHP and MySQL (XAMPP) that allows users to rent cars and bikes through an interactive dashboard.

## Features
- Separate sections for Cars and Bikes on the dashboard.
- User Registration and Login system.
- Users can select, enter details, and book vehicles easily.
- Data stored securely in a MySQL database.
- Simple and responsive UI for smooth navigation.

## Technologies Used
- Frontend: HTML, CSS  
- Backend: PHP  
- Database: MySQL  
- Server Environment: XAMPP

## Setup Instructions
1. Clone or download this repository.
2. Move the project folder to htdocs inside your XAMPP installation.
3. Start Apache and MySQL from XAMPP Control Panel.
4. Import the provided SQL file into phpMyAdmin.
5. Open the project in your browser at:
   http://localhost/your_project_name

## Dashboard Overview
- Cars Section: Displays available cars for rent.  
- Bikes Section: Displays available bikes for rent.  
- Booking Page: Users enter personal and rental details to confirm booking.
